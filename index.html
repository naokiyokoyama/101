Part 1: Blink Program


// Declare a pin to be an OUTPUT or INPUT
pinMode(pin, mode);
// 'pin' must be between 0-13
// 'mode' must be INPUT or OUTPUT

// Change the state of an OUTPUT pin
digitalWrite(pin, state);
// 'pin' must be between 0-13
// 'state' must be HIGH or LOW

// Wait before exceuting the next line of code
delay(time);
// 'time' is a positive integer in MILLISECONDS
